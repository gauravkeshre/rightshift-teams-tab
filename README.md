# Right Shift

A simple react application for Microsoft Teams. This app allows user to login using SSO and log attendence."

## Main features

1. You can side load into Microsoft teams. 
2. Login using SSO.
3. Log attendence.
4. View attendence.
5. View your profile.


## Stack

- indexDb
- express
- react
- Microsoft Graph API
- Microsoft Teams SDK
- Azure AD

## How to run

```bash
npm install 
npm run start

```
